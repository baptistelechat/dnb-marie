# BDR-035 — Frise verticale : axe AXIS_X=380, labels multi-lignes gauche, périodes droite

| Décision                                                                                                                                                                                                                                                                      | Pourquoi                                                                                                                                                                                                                                                | Alternatives considérées                                                                                                                             | Statut |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Axe vertical à x=380, labels points à gauche en `text-align: right` format "DATE — Intitulé" multi-lignes (`whiteSpace: normal`), barres périodes à droite avec `writing-mode: vertical-rl`.
