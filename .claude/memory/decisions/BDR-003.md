# BDR-003 — shadcn/ui : différé jusqu'au besoin réel de composants complexes

| Décision                                  | Pourquoi                                                                                               | Alternatives considérées                                            | Statut |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------- | ------ |
| Ne pas installer shadcn/ui pour l'instant | Aucun composant actuel (Dialog, Select, Toast…) ne le justifie — Tailwind pur + Lucide React suffisent | Installation immédiate (rejeté — surcharge pour des cartes simples) | actif  |

## Déclencheur d'installation

Installer shadcn/ui dès qu'un de ces besoins émerge :

- Modal/Dialog avec gestion focus et Escape
- Select/Combobox accessible
- Toast/Notification système
- Form avec validation Zod + champs complexes
