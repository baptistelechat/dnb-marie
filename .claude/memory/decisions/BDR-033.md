# BDR-033 — Sources photos non-Wikimedia acceptées pour l'app privée

| Décision                                                                                                                                                                                                                                                           | Contexte                                                                                                                                                                        | Alternatives écartées                                                                                                                                               | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Pour les personnages historiques dont les portraits sont absents de Wikimedia Commons (Élizé, Geneviève de Gaulle, Tillion), toute source en ligne est acceptable — outremermemory.com, cheminsdememoire.gouv.fr, museedelaresistanceenligne.org, lalibre.be, etc. | L'app dnb-marie est privée, destinée à une seule utilisatrice (Marie), sans distribution publique ni hébergement public accessible. Aucune contrainte de licence ne s'applique. | Se limiter à Wikimedia Commons : Élizé n'aurait aucune photo (article Wikipedia FR marqué "à illustrer"), Tillion et Geneviève auraient des portraits insuffisants. | actif  |

## Mise en garde

L'URL La Libre (`lalibre.be/resizer/v2/...?auth=...`) contient un token d'authentification potentiellement éphémère. Si la photo d'Élizé disparaît, remplacer par l'URL outremermemory.com qui est stable.
