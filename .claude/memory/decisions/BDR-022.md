# BDR-022 — Replace-in-place `array.map(c => c === target ? next : c)` pour stabilité positionnelle

| Décision                                                                                                                                                                                                                                               | Pourquoi                                                                                                                                                                                                                                | Alternatives considérées                                                                                                                                | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Dans `updateBoardAfterMatch`, quand une paire est validée et que le pool n'est pas vide, le nouvel item **remplace en place** l'item retiré dans `display`, `leftOrder` et `rightOrder` via `array.map(c => c === matchedCode ? next.countryCode : c)` | Bug signalé par Baptiste : "il arrive parfois que la position des boutons change, c'est perturbant." La version initiale utilisait `filter` (supprime) + insert aléatoire → les items voisins se décalaient visuellement à chaque match | `filter` + `push` en fin de tableau (déplace la cible, perturbe la mémorisation) ; shuffler les colonnes après chaque match (encore plus déstabilisant) | actif  |

## Code

```typescript
// Pool non vide : remplace en place
const next = newPool.pop()!;
return {
  pool: newPool,
  display: prev.display.map((p) => (p.countryCode === matchedCode ? next : p)),
  leftOrder: prev.leftOrder.map((c) =>
    c === matchedCode ? next.countryCode : c,
  ),
  rightOrder: prev.rightOrder.map((c) =>
    c === matchedCode ? next.countryCode : c,
  ),
};

// Pool vide : filter simple (la colonne rétrécit en fin de partie)
return {
  display: prev.display.filter((p) => p.countryCode !== matchedCode),
  leftOrder: prev.leftOrder.filter((c) => c !== matchedCode),
  rightOrder: prev.rightOrder.filter((c) => c !== matchedCode),
};
```
