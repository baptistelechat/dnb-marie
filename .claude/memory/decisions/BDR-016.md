# BDR-016 — Layout App `h-dvh flex-col` conditionnel — actif uniquement sur l'onglet Carte Quiz

| Décision                                                                                                                 | Pourquoi                                                                                                                                                            | Alternatives considérées                                                                                                                      | Statut |
| ------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| App outer div : `h-dvh flex-col` si `activeTab === "map-quiz"`, sinon `min-h-dvh`. `#root` revenu à `min-height: 100dvh` | Appliquer `h-dvh flex-col` globalement force un scroll élément sur tous les onglets (Liste, Flashcards) → scrollbar OS visible + perte du scroll navigateur naturel | `overflow: hidden` sur `<main>` partout (non viable — coupe le contenu) ; media query responsive (inefficace sans la logique React activeTab) | actif  |
