# BDR-052 — Faux positifs react-doctor documentés dans `.react-doctor/false-positives.md`

| Décision                                                                                                | Pourquoi                                                                                                                                   | Alternatives considérées                         | Statut |
| ------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------ | ------ |
| Documenter les faux positifs dans `.react-doctor/false-positives.md` plutôt que les silencer via config | Permet de re-vérifier à chaque run si le FP est toujours valide — la config reste propre et les vraies occurrences ne sont jamais masquées | Ignorer via doctor.config (perte de traçabilité) | actif  |
