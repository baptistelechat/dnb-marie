# BDR-017 — DOM-TOM intégrés comme overlay CSS absolu dans la zone océan du canvas SVG principal

| Décision                                                                                                                                                                   | Pourquoi                                                                                                                                                           | Alternatives considérées                                                                                                                             | Statut |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Canvas SVG principal élargi à `1000×700` pour créer une zone vide à gauche (océan Atlantique), puis 5 mini-cartes DOM positionnées via `position:absolute` dans cette zone | Reproduit fidèlement les cartes officielles France avec DOM en incrustés dans l'espace marin — pas d'extension du layout horizontal, pas de colonne supplémentaire | Colonne latérale séparée (manque d'authenticité) ; DOM inline dans le SVG principal (projections incompatibles avec des territoires ultra-lointains) | actif  |

## Détails

Le wrapper de la carte a `position:relative`. L'overlay DOM a `position:absolute inset-y-0 left-0 flex flex-col justify-center items-end` avec `width: 27%` — calé sur la position de la côte ouest française dans le canvas (27.2% après centrage à `center_lon=0.44°`).
