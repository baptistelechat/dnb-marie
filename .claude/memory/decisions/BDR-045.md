# BDR-045 — HintButton inline avec "Valider" : `flex-wrap` + `basis-full order-last`

| Décision                                                                                                                                                                | Pourquoi                                                                                                                                               | Alternatives considérées                                                           | Statut |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------- | ------ |
| HintButton dans le même `flex-wrap` que "Valider" — non révélé : inline en ligne, révélé : wrapper `basis-full order-last` le pousse sur sa propre ligne pleine largeur | Rendre "Proposition" visible dès le premier regard sans imposer une rangée séparée permanente ; les choix révélés occupent naturellement plus d'espace | Row toujours séparée — moins prominent ; Popover — trop complexe pour cible mobile | actif  |
