# BDR-048 — `PhotoCard` : `object-contain` + hauteur fixe 300px

| Décision                                                                                             | Pourquoi                                                                                                                            | Alternatives considérées                                                                                                       | Statut |
| ---------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ | ------ |
| `object-contain` + `height: 300px` + background neutre `#f5f0fb` dans un conteneur `overflow-hidden` | `object-cover` + `aspectRatio: "1/1"` coupait les portraits historiques en haut/bas (ex. photo Hitler : seulement le torse visible) | `object-cover` + `aspect-square` (rejeté : crops des visages), hauteur responsive (rejeté : incohérence visuelle entre cartes) | actif  |
