## Décision

Ajout d'un toggle switch binaire dans `EuropTab` contrôlé par `useState<boolean>(false)`.
La prop `showCapitalFirst` est propagée à chaque `CountryCard`.

Dans `CountryCard`, deux variables locales :

```ts
const primary = showCapitalFirst ? capital : name;
const secondary = showCapitalFirst ? name : capital;
```

Le texte primaire (bold, `text-xs`, opacity pleine) et secondaire (`text-[10px]`, opacity 0.6) s'inversent sans duplication de composant.

Le switch UI positionné entre le header et la progress card : labels "Pays" / "Capitales" colorés selon l'état actif (violet `#9575cd`) vs inactif (slate `#94a3b8`), pastille blanche qui glisse via `translateX`.

## Motivation

Permet deux modes d'étude distincts depuis la même vue :

- **Pays** : entraîner la reconnaissance drapeau → pays
- **Capitales** : entraîner la reconnaissance drapeau → capitale

## Impact

- `EuropTab/index.tsx` : +1 state, +1 bloc JSX toggle
- `EuropTab/components/CountryCard.tsx` : +1 prop, +2 variables locales, affichage inchangé
