# BDR-040 — Tooltip rendu dans GameTimeline (parent), pas dans DropZone

| Décision                                                                                                               | Pourquoi                                                                                                      | Alternatives considérées                                                   | Statut |
| ---------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- | ------ |
| `hoveredId` state dans GameTimeline, callbacks `onHoverStart`/`onHoverEnd` dans DropZone, tooltip rendu dans le parent | DropZone range a `overflow: hidden` pour contraindre le texte vertical → tooltip clipé si rendu dans l'enfant | `overflow: visible` sur DropZone + div interne pour le clip, portail React | actif  |
