# BDR-009 — Mode dirigé : continue jusqu'à 100% + requeue aléatoire + score "premier essai"

| Décision                                                                                                                                                                                                                            | Pourquoi                                                                                                                                                   | Alternatives considérées                                                | Statut |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- | ------ |
| Si réponse correcte → pays retiré de la queue. Si réponse fausse → pays réinséré à position aléatoire (entre index 1 et fin, jamais immédiatement suivant). Fin = queue vide + 27/27. Score affiché : `firstAttemptCodes.size / 27` | Pédagogiquement meilleur pour réviser le DNB — on ne passe pas à côté d'un pays mal appris. Le score "premier essai" donne une métrique honnête sans punir | Option A : fin à 27 passages (même avec erreurs), affichage score final | actif  |

## Requeue aléatoire

```ts
setQueue((prev) => {
  const [current, ...rest] = prev;
  if (rest.length === 0) return [current!];
  const insertAt = Math.floor(Math.random() * (rest.length - 1)) + 1;
  const newQueue = [...rest];
  newQueue.splice(insertAt, 0, current!);
  return newQueue;
});
```
