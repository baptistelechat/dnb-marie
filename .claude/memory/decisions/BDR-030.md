## Décision

L'interface `HistoricalDate` dans `src/data/historicalDates.ts` expose :

```ts
interface HistoricalDate {
  id: string;
  type: "point" | "range";
  date: string; // affichage : "1939" ou "1939–1945"
  sortKey: number; // année de début, pour tri et placement
  endSortKey?: number; // année de fin — obligatoire si type === 'range'
  event: string;
  period?: string;
}
```

Pour `normalizeDateAnswer` : tolérance de ±1 an sur les points, tolérance sur les dates de début ET de fin pour les ranges.

**Why:** Sans distinction `point`/`range`, un intervalle comme "1939–1945" se réduisait à un seul point sur la frise — la durée de la période disparaissait. `endSortKey` permet à la Frise et au jeu Ordonner de rendre la pilule à la bonne longueur. Identifié lors du challenge Rodin.

**How to apply:** Respecter le contrat `endSortKey` obligatoire si `type === "range"` — ajouter une vérification TypeScript si possible (`type HistoricalDateRange = HistoricalDate & { type: 'range'; endSortKey: number }`).
