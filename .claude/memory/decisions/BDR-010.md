# BDR-010 — Markers SVG pour les petits pays EU (BE, LU, SI, MT, CY)

| Décision                                                                                                                                                                             | Pourquoi                                                                                                                                                         | Alternatives considérées                                              | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- | ------ |
| Ajouter `<Marker>` react-simple-maps au centroïde des 5 petits pays EU : cercle `r=6`, même couleurs que le pays (gris/actif/trouvé), cliquable en mode libre, passif en mode dirigé | Ces pays sont trop petits à 50m de résolution pour être cliquables facilement. Le marker garantit une affordance cliquable quelle que soit la taille SVG du pays | Augmenter la résolution (countries-10m) : +3× la taille du bundle geo |

## Coordonnées centroïdes

```ts
const SMALL_EU_MARKERS: Record<string, [number, number]> = {

};
```
