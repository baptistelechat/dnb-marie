# BDR-034 — "Voir le drapeau" tracké comme hintScore dans MapQuizTab

| Décision                                                                                                                                                                                            | Pourquoi                                                                                                                                                                                                                                                            | Alternatives considérées                                                                                                 | Statut |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ | ------ |
| `flagUsedCodes: Set<string>` trace les pays dont le drapeau a été révélé ; `flagUsedCodes.size` = `hintScore` à la sauvegarde.
