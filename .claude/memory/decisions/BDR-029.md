## Décision

Avant d'implémenter le premier jeu du module Histoire (Phase 2), une Phase 1b crée l'infrastructure partagée :

- `HintButton` — génère 3 distracteurs aléatoires + bonne réponse, shuffled. Incrémente `hintScore` côté parent.
- `GameOverModal` — unifié avec `freeScore`/`hintScore`/`missedScore`/`time` (remplace les modals existantes qui n'ont que `firstTryScore`)
- `useLeaderboard` — contrat mis à jour : `{ freeScore, hintScore, missedScore, time, name, date }`

La Phase 10 (retrofit UE/France) branche simplement le `HintButton` déjà construit.

**Why:** Identifier l'infrastructure partagée après avoir déjà codé N composants oblige à retrouver tous les sites + risque de régression sur du code "oublié". La Phase 1b garantit que tout est en place avant la première utilisation. Identifié lors du challenge Rodin.

**How to apply:** Créer ces 3 éléments en Phase 1b, avant Phase 2. Ne pas dupliquer la logique HintButton dans chaque composant de jeu.
