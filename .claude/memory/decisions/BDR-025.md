# BDR-025 — `lang="fr" translate="no"` + notranslate meta pour bloquer Chrome auto-translate

| Décision                                                                                         | Pourquoi                                                                                                                                                                                                          | Alternatives considérées                                                                        | Statut |
| ------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- | ------ |
| `<html lang="fr" translate="no">` + `<meta name="google" content="notranslate">` dans index.html | La page était en `lang="en"` (valeur Vite par défaut) → Chrome proposait la traduction et la déclenchait automatiquement sur les libellés React dynamiques (labels AssociationTab) → texte manglé ("slov", "sud") | `translate="no"` seul sans `lang` — insuffisant, Chrome se base d'abord sur `lang` pour décider | actif  |
