# BDR-036 — Tooltip affiché pour toutes les barres de période (pas seulement les petites)

| Décision                                                                                                                              | Pourquoi                                                                                                                                                        | Alternatives considérées                                                                                                               | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Supprimer la condition `!showText` — le tooltip date+événement s'affiche au survol sur toutes les barres quelle que soit leur hauteur | Les grandes barres affichaient l'intitulé vertical mais pas la date (ex : "Staline au pouvoir en URSS" sans "1924–1953"). Sans tooltip, la date était invisible | Date inline verticale dans la barre (testé puis retiré sur demande : trop chargé visuellement) ; label externe horizontal (encombrant) | actif  |
