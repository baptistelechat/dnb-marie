# BDR-008 — Layout MapQuizTab full-width + flex-row desktop

| Décision                                                                                                                                                                                             | Pourquoi                                                                                                                    | Alternatives considérées                                                  | Statut |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------- | ------ |
| `App.tsx` : `max-w-2xl` retiré uniquement pour l'onglet `map-quiz` (condition sur `activeTab`). `MapQuizTab` : `flex-col lg:flex-row` — carte gauche (`flex-1`), panel quiz droite (`w-80 shrink-0`) | La carte a besoin d'espace pour être lisible et cliquable, surtout pour les petits pays. Le panel quiz est une sidebar fixe | Colonne unique (max-w-2xl) : carte compressée, petits pays imperceptibles | actif  |

## Implémentation

```tsx
// App.tsx
<main className={activeTab === "map-quiz" ? "pb-20" : "max-w-2xl mx-auto pb-20"}>

// MapQuizTab/index.tsx
<div className="flex flex-col lg:flex-row gap-4 items-start">
  <div className="flex-1 min-w-0">   {/* carte */}
  <div className="w-full lg:w-80 shrink-0">  {/* panel */}
```
