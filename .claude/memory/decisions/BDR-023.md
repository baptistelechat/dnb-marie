# BDR-023 — `FranceAssociationTab` — variante France de l'onglet Association

| Décision                                                                                                                                                                                                                                                                                                                                                                                                           | Pourquoi                                                                                                                                                                                                                                                                         | Alternatives considérées                                                                                                                                                                                        | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Création de `FranceAssociationTab/` distinct, utilisant `FRENCH_REGIONS` (18 régions métropole + DOM), localStorage `dnb-france-association-leaderboard` séparé, et labels `"Régions"` / `"Chef-lieux"` via les props `leftLabel`/`rightLabel` de `AssociationBoard`. `App.tsx` dispatche par `subject`. `AssociationGameOverModal` rendu générique via prop `totalCountries` (remplace les deux `/27` hardcodés). | L'onglet Association affichait les pays UE quel que soit le sujet sélectionné — `App.tsx` l'instanciait sans condition sur `subject`. Les sous-composants avaient des valeurs hardcodées ("Pays", "Capitales", `/27`) invisibles depuis l'index qui bloquaient la réutilisation. | Passer un prop `mode` à `AssociationTab` (rejet : couplage fort, conditionnel dans le composant) ; un composant unique paramétrable (rejet : logique de sélection des données trop différente pour être propre) | actif  |

## Fichiers créés / modifiés

- **Nouveau** `src/components/FranceAssociationTab/index.tsx` — composant principal France
- **Nouveau** `src/components/FranceAssociationTab/utils.ts` — `initBoard` depuis `FRENCH_REGIONS`, ré-exporte `updateBoardAfterMatch`
- **Nouveau** `src/components/FranceAssociationTab/hooks/useFranceAssociationLeaderboard.ts` — clé `dnb-france-association-leaderboard`
- **Modifié** `AssociationBoard.tsx` — ajout `leftLabel?: string` / `rightLabel?: string` (défaut : `"Pays"` / `"Capitales"`)
- **Modifié** `AssociationGameOverModal.tsx` — ajout `totalCountries: number`, remplacement des `/27` hardcodés
- **Modifié** `App.tsx` — condition `subject === "eu"` pour `AssociationTab`, `subject === "france"` pour `FranceAssociationTab`

## Résultat

18 paires région ↔ chef-lieu (13 métropole + 5 DOM). Leaderboard France isolé du leaderboard UE. Lint 0 erreur, build clean 437 kB.
