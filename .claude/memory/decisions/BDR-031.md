## Décision

L'interface `HistoricalFigure` expose :

```ts
interface HistoricalFigure {
  id: string;
  name: string;
  primaryKeyword: string; // provisoire — 1 seul mot, à affiner en Phase 7
  keywords: string[]; // 2-4 mots-clés par personne, définis en Phase 7
  photos: string[]; // URLs Wikimedia Commons, sourcés en Phase 1
  period: string;
  role: string;
}
```

Les `keywords[]` sont volontairement laissés vides au moment de Phase 1 — ils seront remplis en Phase 7 en s'appuyant sur le cours réel de Marie pour garantir que les associations correspondent à ce qu'elle a appris.

**Why:** Un seul mot-clé ne résume pas un personnage historique. 2-4 mots-clés permettent un jeu d'association plus riche et pédagogiquement pertinent. Remplir à partir du cours de Marie (pas de connaissance générale) garantit la cohérence avec le programme DNB.

**How to apply:** Phase 1 : définir la structure avec `keywords: []` placeholder. Phase 7 : remplir les `keywords[]` avec Marie / à partir du cours, avant d'implémenter le jeu Association Personnages.
