# BDR-002 — Design direction : Fredoka + Nunito, palette candy 6 couleurs cycliques

| Décision                                                                | Pourquoi                                                                              | Alternatives considérées                                                     | Statut |
| ----------------------------------------------------------------------- | ------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- | ------ |
| Fredoka (titres) + Nunito (corps), palette candy cyclique sur 6 teintes | Public cible collégienne — style "aesthetic study" TikTok/Instagram, fun et mémorable | Inter (rejeté — trop générique), shadcn par défaut (rejeté — trop corporate) | actif  |

## Palette candy

6 teintes cycliques assignées par index modulo 6 :

| Index | Couleur | Fond      | Border    |
| ----- | ------- | --------- | --------- |
| 0     | Rose    | `#fce4ec` | `#f48fb1` |
| 1     | Violet  | `#ede7f6` | `#b39ddb` |
| 2     | Bleu    | `#e3f2fd` | `#90caf9` |
| 3     | Mint    | `#e8f5e9` | `#a5d6a7` |
| 4     | Peach   | `#fff3e0` | `#ffcc80` |
| 5     | Lemon   | `#fffde7` | `#fff176` |

## Application

Tout nouvel onglet ou composant de grille doit respecter cette palette. Ne pas introduire de nouvelles couleurs de base sans réviser BDR-002.
