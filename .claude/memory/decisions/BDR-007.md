# BDR-007 — Carte affichée en N&B, pays trouvés en candy-purple-vivid

| Décision                                                                                                                                                                                     | Pourquoi                                                                                                                                                | Alternatives considérées                                     | Statut |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ | ------ |
| Pays EU non trouvés : gris `#c8c8c8` / `#a0a0a0`. Pays trouvés : `#7e57c2` (candy-purple-vivid). Pays actif (sélectionné) : `#ddd6fe` / `#7c3aed`. Pays non-EU : `#e5e7eb` (fond contextuel) | Meilleure lisibilité qu'une carte multicolore candy — le progrès est immédiatement visible (le violet "envahit" la carte au fil des réponses correctes) | Candy colors cycliques (état initial), vert pour les trouvés | actif  |
