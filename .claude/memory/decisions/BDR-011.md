# BDR-011 — Timer démarre au clic sur "GO !" (pas à la première réponse)

| Décision                                                                                                                                                                                     | Pourquoi                                                                                                                               | Alternatives considérées                                                                                                                                 | Statut |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| En mode dirigé, un bouton "GO !" remplace le QuizPanel tant que `timerStartedAt === null`. Le chrono démarre uniquement au clic sur GO — `handleGo` appelle `setTimerStartedAt(Date.now())`. | Contrôle total sur le démarrage : Marie peut consulter la carte avant d'être chronométrée. Le GO est l'acte intentionnel de démarrage. | Démarrer au premier submit → chrono peut démarrer par accident avant d'être prêt. Démarrer au changement de mode → pénalise la consultation de la carte. | actif  |

## Implémentation

```tsx
// handleGo dans MapQuizTab et FranceMapQuizTab
const handleGo = useCallback(() => {
  setTimerStartedAt(Date.now());
}, []);

// JSX : bouton GO remplace QuizPanel avant le GO
{mode === "directed" && timerStartedAt === null ? (
  <div ...>
    <button onClick={handleGo}>GO !</button>
  </div>
) : (
  <QuizPanel ... />
)}
```
