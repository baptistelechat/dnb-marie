# BDR-042 — AssociationBoard row-by-row layout pour synchronisation hauteurs colonnes

| Décision                                                                                                               | Pourquoi                                                                                                                                                        | Alternatives considérées                                                              | Statut |
| ---------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- | ------ |
| Passer de deux `flex-col` indépendants à un `flex-col` de rows `grid-cols-2 items-stretch` + `h-full` sur `ColumnItem` | Les événements historiques (gauche) passent sur 2 lignes, les dates (droite) restent sur 1 ligne — deux flex-col indépendants ne synchronisent pas les hauteurs | `min-height` fixe sur tous les items → gaspillage d'espace ; équilibrage JS → fragile | actif  |

Structure résultante dans `AssociationBoard.tsx` :

- Headers dans un `div grid-cols-2` dédié
- Chaque row : `<div className="grid grid-cols-2 gap-3 items-stretch">` avec `leftOrder[i]` et `rightOrder[i]`
- `ColumnItem` reçoit `className="h-full"` → s'étire à la hauteur du voisin le plus grand
