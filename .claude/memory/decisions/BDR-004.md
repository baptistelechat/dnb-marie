# BDR-004 — react-simple-maps choisi pour la carte EU (SVG pur, React-friendly)

| Décision                        | Pourquoi                                                                             | Alternatives considérées                                                           | Statut |
| ------------------------------- | ------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------- | ------ |
| Utiliser `react-simple-maps` v3 | SVG pur — chaque pays = `<path>` React avec `onClick`, styles inline, animations CSS | Leaflet (tile-based, 400 kB, raster — éliminé) · D3-geo (impératif, anti-React 19) | actif  |

## Détails d'intégration

- `ComposableMap` + `Geographies` + `Geography` (composants déclaratifs)
- Données géographiques via world-atlas TopoJSON — voir [BDR-005](BDR-005.md)
- Filtrage des 27 pays UE par code ISO numérique (voir `constants.ts`)
- Pas de types bundlés → déclaration custom `src/types/react-simple-maps.d.ts` — voir [LRN-002](../learnings/LRN-002.md)
