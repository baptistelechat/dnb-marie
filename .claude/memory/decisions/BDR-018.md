# BDR-018 — Projections DOM-TOM individuelles (center, scale, width) calculées depuis bounding box GeoJSON

| Décision                                                                                                                                                                            | Pourquoi                                                                                                                                                                      | Alternatives considérées                                                                                                             | Statut |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ | ------ |
| Chaque DOM a ses propres `center`, `scale`, `width` stockés dans `DOM_PROJECTION_CONFIG` — valeurs calculées depuis le bounding box exact extrait du GeoJSON + formule mathématique | Valeurs uniformes impossibles : Guyane (7°× 4°) et Mayotte (0.5°× 0.5°) nécessitent des scales totalement différents. La Réunion avec `scale` trop élevé débordit du viewport | Scale uniforme (coupe les plus larges) ; zoom purement visuel/empirique (non reproductible, fragilité sur changement de canvas size) | actif  |

## Config finale

```ts
const DOM_PROJECTION_CONFIG = {
  "01": { center: [-61.406, 16.173], scale: 4700, width: 80, height: 70 }, // Guadeloupe
  "02": { center: [-61.019, 14.634], scale: 6600, width: 58, height: 70 }, // Martinique
  "03": { center: [-53.11, 3.93], scale: 950, width: 57, height: 70 }, // Guyane
  "04": { center: [55.527, -21.131], scale: 6000, width: 78, height: 70 }, // La Réunion
  "06": { center: [45.159, -12.821], scale: 8500, width: 53, height: 70 }, // Mayotte
};
```
