# BDR-047 — `buildRoundPool()` : 1 photo aléatoire par personnage par partie

| Décision                                                                                               | Pourquoi                                                                                                                 | Alternatives considérées                                                                                                                          | Statut |
| ------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Sélectionner 1 photo au hasard parmi `fig.photos[]` pour chaque personnage à chaque partie (11 cartes) | 44 photos total = trop long pour une session de révision ; la variété est maintenue car le tirage change à chaque replay | Toutes les 44 photos (rejeté : trop long), sous-ensemble aléatoire de N photos (rejeté : complexité inutile, 11 = nb de personnages déjà correct) | actif  |

## Architecture

```typescript
const buildRoundPool = (): PhotoEntry[] =>
  HISTORICAL_FIGURES.map((fig) => {
    const photoIndex = Math.floor(Math.random() * fig.photos.length);
    return { entryId: `${fig.id}-${photoIndex}`, photoUrl: fig.photos[photoIndex]!, ... };
  });

// Dans le composant :
const [roundPool, setRoundPool] = useState<PhotoEntry[]>(() => buildRoundPool());
const roundMap = useMemo(() => new Map(roundPool.map((e) => [e.entryId, e])), [roundPool]);

// handleReplay : nouveau tirage complet
const newPool = buildRoundPool();
setRoundPool(newPool);
setQueue(shuffle(newPool.map((e) => e.entryId)));
```
