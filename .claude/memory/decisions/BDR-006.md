# BDR-006 — Haptics centralisés dans `useHaptics()` — `cancel()` avant `trigger()` pour taps rapides

| Décision                                                                                                 | Pourquoi                                                                                                                                                                                                 | Alternatives considérées                                                               | Statut |
| -------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- | ------ |
| Créer `src/utils/hapticPatterns.ts` avec hook `useHaptics()` exposant `tick()` / `success()` / `error()` | Bug : `navigator.vibrate()` ignoré sur Android si vibration déjà en cours → `cancel()` obligatoire avant chaque `trigger()` dans `tick()`. Centralisation des patterns pour uniformité cross-composants. | Appels directs `trigger()` dispersés dans chaque composant (état initial, sans cancel) | actif  |

## Patterns définis

```ts
// tick — tap rapide (toggle pays, soumission quiz)
cancel();
trigger(40);

// success — double pulse doux → fort
trigger([{ duration: 30 }, { delay: 60, duration: 40, intensity: 1 }]);

// error — quatre impulsions progressives
trigger([
  { duration: 40, intensity: 0.7 },
  { delay: 40, duration: 40, intensity: 0.7 },
  { delay: 40, duration: 40, intensity: 0.9 },
  { delay: 40, duration: 50, intensity: 0.6 },
]);
```

## Règle d'usage

- `tick()` : toute action rapide répétable (toggle, validation). Le `cancel()` interne évite les pertes sur taps < 40 ms.
- `success()` : complétion de session (27/27 pays, 27/27 quiz).
- `error()` : réponse incorrecte (non encore câblé en 2026-06-03).
