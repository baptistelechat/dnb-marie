# LRN-023 — Centrage vertical d'un tooltip sur un élément de hauteur variable

| Pattern observé                                                                                                                                   | Contexte                                                                                                                                           | Application future                                                                                     |
| ------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| `top: elementTop + elementHeight/2` + `transform: translateY(-50%)` centre un tooltip absolu sur un élément à hauteur quelconque sans recalcul JS | Barres de période (height = durée × PX_PER_YEAR), tooltip ancré à `top: y + barHeight/2` ; centrage correct quelle que soit la durée de la période | Tout tooltip absolute devant s'aligner au milieu d'un sibling à hauteur dynamique (card, barre, image) |
