# LRN-001 — `vite-plugin-qrcode` requiert `server.host: true`

| Pattern observé                                                                                                                | Contexte                                             | Application future                                                                                  |
| ------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| Sans `server.host: true`, Vite n'expose qu'une URL localhost — pas d'URL réseau → le QR code ne s'affiche pas dans le terminal | Setup dev server pour accès mobile LAN sur dnb-marie | Toujours ajouter `server: { host: true }` dans vite.config.ts quand vite-plugin-qrcode est installé |
