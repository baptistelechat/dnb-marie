# LRN-028 — Leaderboard dupliqué sans composant partagé → drift silencieux

| Pattern observé                                                                                                                                                   | Contexte                                                                                                                                                | Application future                                                                                                                                                                                      |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Deux affichages du même leaderboard (modal + écran d'accueil) implémentés en code inline distinct → une feature ajoutée dans l'un n'est pas reportée dans l'autre | `FriseOrdonnnerTab` : `OrdonnerGameOverModal` avait `formatAge`, le leaderboard d'accueil de `index.tsx` non — visible uniquement au retour utilisateur | Dès qu'un classement s'affiche en deux endroits, extraire un composant partagé (`OrdonnerLeaderboardRow`, `LeaderboardPanel`) — les autres jeux utilisent `<LeaderboardPanel>` et n'ont pas ce problème |
