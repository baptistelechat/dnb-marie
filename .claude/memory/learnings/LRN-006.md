# LRN-006 — `<Marker>` react-simple-maps pour les petits pays difficilement cliquables

## Pattern observé

`react-simple-maps` expose un composant `<Marker coordinates={[lon, lat]}>` qui positionne un élément SVG à des coordonnées géographiques fixes. Utile pour les petits pays (Luxembourg, Malte, Chypre) dont la surface SVG à 50m est trop petite pour être cliquable.

## Usage

```tsx
import { Marker } from "react-simple-maps";

// Après </Geographies>, avant </ComposableMap>
{
  Object.entries(SMALL_EU_MARKERS).map(([alpha2, coords]) => (
    <Marker key={`marker-${alpha2}`} coordinates={coords}>
      <circle
        r={6}
        fill={fill} // même couleur que le Geography du pays
        stroke={stroke}
        strokeWidth={1.5}
        style={{ cursor: isInteractive ? "pointer" : "default" }}
        onClick={isInteractive ? () => onSelect(alpha2) : undefined}
      />
    </Marker>
  ));
}
```

## Avantages

- Taille fixe indépendante du zoom et de la résolution des geodata
- Peut avoir les mêmes états visuels (gris / actif / trouvé / pulsant) que le `Geography`
- Pas besoin de changer la résolution des geodata (50m → 10m = +3× le bundle)
