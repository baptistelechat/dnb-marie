# LRN-045 — dev-browser OOM via pipe PowerShell : écrire dans fichier tmp via Bash

| Pattern observé                                                                                                                                                                                                                                      | Contexte                                                   | Application future                                                                            |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| Piper un heredoc PowerShell (`@'...'@`) vers `npx dev-browser` provoque un crash OOM. Fix : écrire le script dans un fichier temporaire via Bash (`cat > /tmp/script.js <<'EOF' ... EOF`), puis exécuter via `cat /tmp/script.js \| npx dev-browser` | Screenshots dev-browser lors de la session FriseLectureTab | Toujours écrire les scripts dev-browser dans un fichier via Bash — jamais via pipe PowerShell |
