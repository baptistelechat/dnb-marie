# LRN-030 — `flex-wrap + basis-full order-last` = expanded element pushed to own full-width row

| Pattern observé                                                                                                                                                                                                                                             | Contexte                                                                                                                                             | Application future                                                                                                                |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| Wrapper `div` avec `className={expanded ? "basis-full order-last" : ""}` dans un parent `flex flex-wrap gap-2` : quand `basis-full`, l'élément occupe 100% et force un retour à la ligne ; `order-last` le place après les autres quel que soit l'ordre DOM | HintButton qui révèle 4 choix — inline avec "Valider" avant révélation, pleine largeur en dessous après révélation, sans modifier le reste du layout | Tout bouton ou zone qui change de taille contextuelle dans une toolbar flex — notifications, accordéons inline, zones de feedback |
