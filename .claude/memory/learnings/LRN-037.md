# LRN-037 — react-doctor : 3 catégories de faux positifs sur ce projet

| Pattern observé                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | Contexte                                           | Application future                                                                                                                  |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
