# LRN-032 — `object-contain` + hauteur fixe = portrait entier visible

| Pattern observé                                                                                                                                                   | Contexte                                                                                                                                       | Application future                                                                                                                                 |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| `object-cover` + `aspectRatio` force un recadrage qui coupe les sujets ; `object-contain` + `height` fixe garde l'image entière avec du fond neutre sur les côtés | `PhotoCard` — photos de personnages historiques avec ratios variés (portrait, paysage, carré) : `object-cover aspect-square` coupait les têtes | Toute galerie ou flashcard affichant des portraits avec des ratios inconnus — préférer `object-contain` quand le sujet doit être visible en entier |
