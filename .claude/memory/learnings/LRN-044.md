# LRN-044 — dev-browser : `browser.newPage()` timeout → utiliser `browser.getPage("main")`

| Pattern observé                                                                                                                                                                                                                                                                | Contexte                                                                              | Application future                                                                            |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `browser.newPage()` dans les scripts dev-browser attend un contexte qui ne se ferme jamais et timeout systématiquement (30 s). `browser.getPage("main")` récupère la page déjà ouverte dans le daemon.
