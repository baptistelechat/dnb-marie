# LRN-029 — 2 colonnes hauteurs variables → rows `grid-cols-2 items-stretch + h-full`

| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| Deux `flex-col` indépendants ne synchronisent pas les hauteurs quand l'une a du texte multi-ligne et l'autre du texte court.
