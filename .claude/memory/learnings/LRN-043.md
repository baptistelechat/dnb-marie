# LRN-043 — `overflow: hidden` flex ne clippe pas `writing-mode: vertical-rl`

| Pattern observé                                                                                                                                                                                                           | Contexte                                                                                                                   | Application future                                                                                                              |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `overflow: hidden` sur un conteneur `flex` ne clippe pas fiablement un enfant avec `writing-mode: vertical-rl`. Fix : wrapper `<div style={{ height: N, overflow: "hidden" }}>` avec hauteur explicite autour du `<span>` | Débordement de texte vertical dans les barres de période `TimelineCanvas.tsx` malgré `overflow: hidden` sur le parent flex | Toujours wrapper le contenu vertical dans un `<div>` sized avant `overflow: hidden` — ne jamais compter sur le parent flex seul |
