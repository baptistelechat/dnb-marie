# LRN-041 — Pattern URL CDN stable : portraits officiels présidents français (Élysée)

| Pattern observé                                                                                                                                                                                                                                                         | Contexte                                                                                             | Application future                                                                                                     |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Tous les présidents de la Ve République ont un portrait officiel accessible sur `elysee.fr` via le CDN Cloudflare Images, avec le pattern `https://www.elysee.fr/cdn-cgi/image/format=auto%2Cquality=100%2Cwidth=1520%2Cheight=2534/images/default/0001/02/[hash].jpeg` | Intégration des portraits de Pompidou, Giscard, Mitterrand, Chirac, Sarkozy, Hollande dans dnb-marie | Tout ajout de président Ve République dans historicalFigures : chercher le portrait officiel sur elysee.fr en priorité |
