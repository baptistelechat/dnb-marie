# LRN-017 — WordPress thumbnail `-WxH` → full-res par suppression du suffixe

## Pattern

WordPress génère automatiquement des thumbnails en ajoutant un suffixe de dimension dans le nom de fichier avant l'extension :

```
https://example.com/wp-content/uploads/2021/02/Raphael-2-199x300.png  ← thumbnail
https://example.com/wp-content/uploads/2021/02/Raphael-2.png           ← original
```

Supprimer le suffixe `-WxH` donne l'URL de l'image originale pleine résolution.

## Comment appliquer

1. Repérer le suffixe `-\d+x\d+` avant l'extension
2. Tester la version sans suffixe via HEAD — si 200 → utiliser l'original
3. Si 404 → l'original n'est pas exposé, utiliser le thumbnail

## Cas rencontré

`Raphael-2-199x300.png` → `Raphael-2.png` (HEAD 200 confirmé) — portrait pleine résolution d'Élizé.
