# LRN-038 — HTML standalone = gate de validation avant intégration d'assets externes

| Pattern observé                                                                                                                                                                                                                           | Contexte                                                                                                                                                                                 | Application future                                                                                                                                                                                                                |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Un fichier HTML standalone (sans bundler, sans serveur) peut servir de gate de validation manuelle pour des ressources externes — l'utilisateur coche ce qui passe, exporte le résultat en JSON, et seulement ce résultat va dans le code | `src/data/validate-photos.html` dans dnb-marie : affiche les URLs candidates, détecte les 404 visuellement (`img.onerror`), permet la sélection par clic, exporte en JSON via une modale | Tout projet avec assets externes non contrôlés (URLs Wikimedia, CDN tiers, assets uploadés) — créer un outil HTML minimal de curation avant d'injecter dans la source. Le STOP doit être explicite dans le guide de contribution. |
