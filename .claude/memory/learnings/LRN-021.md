# LRN-021 — Anti-collision greedy push-down pour événements sub-année sur axe vertical

| Pattern observé                                                                                                                                                                            | Contexte                                                                                                                                                                                                                        | Application future                                                                                       |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Trier les événements par `sortKey + (moisFrançais − 0.5) / 12`, puis itérer avec `y = Math.max(naturalY, lastY + MIN_SPACING)` pour ne pousser vers le bas que les événements trop proches | Août 1914/Nov 1918, Juin/Oct 1940, Jan/Mai/Août 1945 — même année → chevauchement à 22px naturel. `MIN_SPACING=18 < PX_PER_YEAR=22` : les événements d'années différentes ne cascadent pas, seuls les sub-annuels sont corrigés | Toute frise verticale avec événements simultanés ou quasi-simultanés : calendrier, axe historique, Gantt |
