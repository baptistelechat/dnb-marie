# LRN-035 — HintButton inline-first : wrapper sans classe → `basis-full order-last` après révélation

| Pattern observé                                                                                                                                                                                                                                  | Contexte                                                                                                                                                                                                      | Application future                                                                                                                                                                                                                                   |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `<div className={hintRevealed ? "basis-full order-last" : ""}>` — sans classe = inline avec Valider dans le flex-wrap. `basis-full` déclenché uniquement après clic "Proposition" pour pousser les 4 choix sur leur propre ligne pleine largeur. | Phase 10 : branchement HintButton dans `CapitalsQuizTab` et `FranceCapitalsQuizTab`. Erreur initiale : wrapper `basis-full` inconditionnel → bouton seul sur sa ligne dès le départ, avant toute interaction. | Tout bouton "expand-on-click" dans un flex-wrap : partir de l'état compacté (inline), s'expander seulement après activation. Anti-pattern à éviter : appliquer le layout expanded dès le rendu initial. Complète [LRN-030](../learnings/LRN-030.md). |
