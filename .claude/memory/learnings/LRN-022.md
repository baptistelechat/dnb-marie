# LRN-022 — `whiteSpace: normal` + `MIN_POINT_SPACING` accru = labels lisibles sans scroll

| Pattern observé                                                                                                                                     | Contexte                                                                                                                                            | Application future                                                                                                         |
| --------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| Labels absolus multi-lignes (`whiteSpace: normal`) + espacement minimal accru entre points > `whiteSpace: nowrap` + ellipsis + grand axe (→ scroll) | Zone label `right: calc(100% - 367px)` trop étroite pour les textes longs à 11px sur mobile 375px ; AXIS_X=460 créait un scroll latéral indésirable | Toute frise verticale absolute-positioned sur écran étroit : préférer le wrap + MIN_SPACING adapté plutôt qu'élargir l'axe |
