# LRN-020 — `right: calc(100% - Xpx)` sur `left:0` = zone de largeur fixe depuis le bord gauche

| Pattern observé                                                                                                                              | Contexte                                                                                                                                                  | Application future                                                                                                                                              |
| -------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `position:absolute; left:0; right:calc(100% - ${X}px)` donne un élément de largeur exactement X pixels, quelle que soit la largeur du parent | Labels de timeline devant se terminer à un pixel précis (LABEL_RIGHT_EDGE = AXIS_X − POINT_R − 6 = 287px) par rapport au bord gauche du conteneur relatif | Toute zone de label devant s'arrêter à un pixel fixe dans un conteneur relatif : légendes de graphique, colonnes gauche en layout absolue, overlays positionnés |
